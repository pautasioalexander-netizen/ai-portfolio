FinBot —
Autonomous Financial
Assistant

An AI-powered financial assistant built for the Argentine market. FinBot automatically ingests and transcribes financial analysis content, queries real-time portfolio data from a local broker API, and responds to investment queries in natural language.

Built on a local-first architecture with automated cron jobs, a Flask dashboard, and Llama 3.3 70B running via NVIDIA NIM. The system processes 53+ transcriptions from a financial analyst and maintains a live portfolio context.

Currently in MVP phase — the core query loop is functional. Active channel is the local Flask dashboard. IOL broker token requires manual renewal to restore real-time portfolio data.
